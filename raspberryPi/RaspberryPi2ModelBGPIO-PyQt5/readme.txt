In order to run the RaspberryPi2ModelBGPIO.py script, you need to install you 
Python 3 and the you need to have the PyQt5 library as well as the RPi.GPIO 
library. 

Note: This Pin Out Diagram is for the Raspberry Pi 2 Model B, not meant for use 
with other versions.
